# Miamgo 🍽️
Miamgo est une application de livraison de repas rapide et locale au Cameroun, intégrée à l’écosystème Pi Network.

## Fonctionnalités
- Commande de repas
- Paiement avec Pi
- Interface simple pour les utilisateurs camerounais
